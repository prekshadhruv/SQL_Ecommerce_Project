E-commerce Sales Analysis (SQL Project)

Overview

This project analyzes an e-commerce dataset using SQL (Northwind Database) to extract meaningful business insights related to sales, customers, shipping, and revenue trends.

Tools Used

- Microsoft SQL Server (SSMS)
- Northwind Sample Database

Key Analysis Performed

- Monthly Revenue Trend Analysis
-  Top 5 Customers by Total Sales
- Customers with No Orders
- Shipping Performance Analysis
- Country-wise Sales and Revenue Analysis
- Customer Total Spending Analysis
- Year-wise Revenue Analysis

Key Insights

 - Identified top-performing customers contributing maximum revenue
 - Observed monthly and yearly sales trends
 - Detected inactive customers with no orders
 - Compared performance of different shipping partners
 - Analyzed revenue contribution across countries

 Files Included

 `SQL_Ecommerce_Project.sql` → Contains all SQL queries used for analysis

 Conclusion

This project demonstrates strong understanding of SQL concepts including:

- Joins
- Aggregations
- Group By & Having
- Subqueries
- Business-driven analysis
